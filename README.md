About
-----

Furtherfield Theme is based on Neo Blue, distributd with GNU social.


Setup
-----

1. Add the directory furtherfield-social-theme to your /theme/ folder

2. add `$config['site']['theme'] = 'furtherfield-social-theme';` to config.php
